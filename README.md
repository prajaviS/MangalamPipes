# Mangalam HDPE Pipes — Product Page

A responsive product page built with vanilla HTML, CSS, and JavaScript.

## Features
- Sticky header that appears on scroll and hides when scrolling down
- Image carousel with zoom lens preview panel
- Interactive manufacturing process tabs
- Horizontally scrollable industry and testimonial carousels
- FAQ accordion
- Contact form with validation
- Fully responsive across desktop, tablet, and mobile

## Setup
Just open `index.html` in a browser — no build tools or dependencies needed.

## Tech Stack
Vanilla HTML5, CSS3, JavaScript — no frameworks or libraries.
